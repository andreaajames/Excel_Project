📊 COVID-19 & RealMart Sales — Excel Analysis Project

This Excel project contains two independent datasets — **Global COVID-19 Statistics** and **RealMart Retail Sales** — along with pivot table analysis and interactive dashboards built entirely in Microsoft Excel.

---

## 📁 File

`Excel_Final_Project_Data_Set__version_1__xlsb.xlsx`

---

## 🗂️ Sheets Overview

| Sheet | Description |
|---|---|
| `Covid19` | Raw COVID-19 data (10,000 records across 5 countries) |
| `RealMart_Sales` | Retail sales transactions (2,064 records, Q1 2019) |
| `analyis` | Pivot table analysis for COVID-19 data |
| `dashboard_covid` | Visual dashboard for COVID-19 insights |
| `dashboard_realmart` | Visual dashboard for RealMart sales insights |
| `Dashboard` | Combined summary dashboard |
| `Sheet1` | Supporting calculations |

## 🦠 Dataset 1: COVID-19 Global Statistics

### Overview
- **Records:** 10,000
- **Countries:** Germany, Brazil, India, UK, USA
- **Period:** April 2020 onwards

### Columns
| Column | Description |
|---|---|
| RecordID | Unique record identifier |
| Country | Country name |
| State | State/region |
| City | City name |
| ReportDate | Date of the report |
| NewCases | New cases reported |
| NewDeaths | New deaths reported |
| NewRecoveries | New recoveries reported |
| TotalCases | Cumulative total cases |
| TotalDeaths | Cumulative total deaths |
| TotalRecovered | Cumulative total recovered |
| TestConducted | Number of tests conducted |
| VaccineDoses | Vaccine doses administered |
| Population | Country population |

## 🛒 Dataset 2: RealMart Sales

### Overview
- **Records:** 2,064
- **Cities:** Hyderabad, Mumbai, Delhi
- **Period:** January – March 2019 (Q1)

### Columns
| Column | Description |
|---|---|
| Invoice ID | Unique invoice number |
| Branch | Store branch (A, B, C) |
| City | City of the branch |
| Customer type | Member or Normal |
| Gender | Customer gender |
| Product line | Category of product sold |
| Unit price | Price per unit |
| Quantity | Units purchased |
| Tax 5% | 5% tax on purchase |
| Total | Total bill amount |
| Payment | Cash / Ewallet / Credit card |
| cogs | Cost of goods sold |
| gross margin percentage | Gross margin % |
| gross income | Gross income from sale |
| Rating | Customer satisfaction rating |
| Date | Transaction date |
| Time / Hour | Transaction time |

## 📈 Dashboards & Analysis

The workbook includes:
- **Pivot tables** summarizing total deaths, recoveries, and cases by country
- **COVID-19 Dashboard** with visual charts and KPIs
- **RealMart Sales Dashboard** showing sales by branch, product line, and payment method
